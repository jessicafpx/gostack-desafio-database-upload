# 🚀 Desafio GoStack: Upload para o banco de dados
Este é um desafio do bootcamp GoStack, da Rocketseat. Trata-se de uma aplicação de gestão de transações que armazena transações financeiras de entrada e saída e permite o cadastro e a listagem dessas transações, além de permitir a criação de novos registros no banco de dados a partir do envio de um arquivo csv.

## ⚙️ Tecnologias
- Node.js
- Express
- TypeScript
- TypeORM
- Multer
